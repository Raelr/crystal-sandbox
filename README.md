## Postgres - Local Machine

Please remember to turn on your Postgres server before trying this!

Commands:

```
# Start the server
postgres -D <PATH_TO_SERVER_DATA>
# Stop the server
pg_ctl -D <PATH_TO_SERVER_DATA> stop -s -m fast
```
